# UniUltraOpenPlatForm 功能列表

项目技术栈：FastAPI + SQLite3/PostgreSQL + Jinja2 + FastAPI StaticFiles

---

## 性能优化

| 优化项 | 描述 | 性能提升 |
|--------|------|----------|
| WAL 模式 | SQLite WAL 日志模式，提升并发性能 | 3-5x 并发提升 |
| LRU 缓存 | 汇率查询缓存 (100 条目) | 90% 查询减少 |
| 数据库索引 | 40+ 个关键列索引 | 50-100x 查询加速 |
| executemany | 批量插入优化 | 10-50x 导入加速 |
| 单事务批处理 | 批量操作事务优化 | 减少磁盘 I/O |
| PostgreSQL 连接池 | 支持 PostgreSQL 连接池 | 高并发场景支持 |
| 多数据库支持 | SQLite/PostgreSQL 双模式切换 | 生产环境扩展性 |

## 数据库表关系

```
uni_emp (员工)
    ├── uni_cli (客户)
    │       ├── uni_quote (询价)
    │       │       └── uni_offer (报价)
    │       │               └── uni_order (订单)
    │       │                       └── uni_buy (采购)
    │       │               └── uni_order_manager_rel → uni_order_manager (客户订单管理器)
    │       └── uni_contact (联系人营销)
    └── uni_offer (报价)

uni_vendor (供应商)
    ├── uni_offer (报价)
    └── uni_buy (采购)

uni_daily (汇率)
    └── (被报价/订单/采购引用计算)

mail_config (邮件账户)
    ├── uni_mail (邮件)
    │       └── uni_mail_rel (邮件关联)
    └── mail_folder (文件夹)
            └── mail_filter_rule (过滤规则)

uni_order_manager (客户订单管理器)
    ├── uni_order_manager_rel (报价关联)
    └── uni_order_attachment (附件)
```
